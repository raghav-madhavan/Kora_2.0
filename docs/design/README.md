# Kora Design Prototype (legacy)

Standalone Vite + Tailwind prototype used during early UI exploration. The canonical product UI now lives in [`apps/web`](../../apps/web).

> **Note:** This folder was originally scaffolded as "ServeTrack." The product is now **Kora**. Treat files here as historical design reference only — do not use the `servetrack` package name in new work.

## Contents

- `DESIGN.md` — design tokens and component notes (some labels still say ServeTrack)
- `src/` — static HTML/JS mock screens
- `design-extract-output/` — Dribbble reference extraction artifacts

## Running (optional)

```bash
cd docs/design
npm install
npm run dev
```

For current development, use `npm run dev` from the [repo root](../../README.md).
